# RGB Feedback

*Max 8.3*

This Jitter Effect displaces a plane of a texture with feedback using jit.gl.pix and codebox.<br/> 
The controls are:<br/>
**feedback**<br/>
**color**: which color(rgb) should be displaced<br/>
**displacement** of the plane<br/>
